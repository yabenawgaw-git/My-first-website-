# NEXORA AI

NEXORA AI is a frontend-only web project that demonstrates a creative AI platform UI for generating **image, video, and game ideas**.

This repository contains only static frontend files and is ready to deploy on **Vercel**.

## Features

- Login / Signup UI (demo)
- Dashboard preview
- Image, Video, and Game idea input
- Clean and responsive UI
- Works on PC and Mobile
- Frontend-only (no backend)

---

## Deployment (Vercel)

1. Upload this repository to GitHub
2. Go to https://vercel.com
3. Click **New Project**
4. Import the GitHub repository
5. Framework Preset: **Other**
6. Root Directory: `./`
7. Build Command: *(leave empty)*
8. Output Directory: `.`
